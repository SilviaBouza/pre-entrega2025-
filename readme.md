# Mi Sitio Web - Ejemplo de Estructura, Estilos y Responsividad

Descripcion:

Este proyecto fue creado con fines educativos para demostrar:
- Estructura semántica en HTML5.
- Uso de CSS externo con fuentes de Google Fonts.
- Diseño responsivo con Flexbox, Grid y Media Queries.
- Formulario de contacto funcional utilizando Formspree.
- Inclusión de contenido multimedia e interacción básica.
